# 🌍 Country Podcast

Daily podcast exploring one country per day — history, politics, conflicts, and regional context.

## About

Each episode provides a ~15-minute overview of a country:
- Geographic location and neighbors
- Key historical events
- Current political situation
- Regional conflicts and relations
- Economic and cultural context

## Source

All content is sourced from Wikipedia and generated using text-to-speech. Scripts are written to be TTS-friendly (no abbreviations, numbers spelled out, natural transitions).
